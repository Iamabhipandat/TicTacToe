# Tic Tac Toe Game (Core Java)

A simple **Tic Tac Toe** console-based game built using **Core Java**.  
I made this project to revise and strengthen my Java fundamentals like loops, conditions, arrays, and methods.

---

## Features
- 2 Player Mode (Player X vs Player O)
- Console-based UI
- Validates input (prevents invalid moves)
- Detects:
  -  Win
  -  Draw
- Simple and clean logic

---

##  Tech Stack
- **Language:** Java (Core Java)
- **IDE:** IntelliJ IDEA
- **Type:** Console Application
